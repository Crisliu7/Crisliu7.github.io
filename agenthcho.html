Agent HCHO

Abstract

Agent HCHO is a portable smart formaldehyde tester. The WIFI chip can send the formaldehyde concentration data detected by sensor to Aliyun Server. The mobile app can read the real-time data and give an alert when exceeding the limit

Product Features
  • Portable
    The product is very small(40mm*35mm*9mm). It get rid of the traditional display screen(instead displayed on mobile phone). Also, the hook on the head make it convenient to take everywhere.
  • Remote Monitoring
    The Wifi chip based on NodeMCU make Agent HCHO automatedly detect and upload the concentration data. The mobile phone can display real-time data graph and predict the formaldehyde concentration in the future.
  • Energy Saving
    Both of The sensor and chip is low-power consumption. Instead of built-in battery, the Agent HCHO uses power bank to charge and is suitable for Micro-USB interface
  • Data Sharing (to be developed)
    The Agent HCHO can acquire the formaldehyde data in any location which have been uploaded by other users'(like a data map)

The model is 3D printed using photosensitive resin, lighter and more environment friendly. The vent is behind the glasses, ensuring air convection. The hook is suitable in multi-Scenarios. The USB interface is at the back.

The data communication model(using MQTT protocol).
